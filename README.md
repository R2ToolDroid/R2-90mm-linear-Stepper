# R2-90mm-linear-Stepper
 4 phasen mini Stepper Controlunit
 mit Motortreiber: HG7881 (L9110) und zwei Taster
 
# Pin Assign:

- A0  Action Switch
- A1  LimitSwitch
- RX  Serial Input    in | out | stop

# Nano to L9110

Stepper Module 
 IN B1 A   11 // 14
 IN B1 B   10 // 15
 IN A1 A   8  // 16
 IN A1 B   9  // 17

 Motor  
 BLK BLU 1
 YEL RED 2
